Recommended way to bootstrap and instantiate a component in typical SAPUI5 app
  • Bootstrap:
    • Recommended script
  • Check SAPUI5 runtime initialization:
    • Via attachInit method
    • Via data-sap-ui-oninit module
    • Via ComponentSupport module(need Component.js and manifest.json)
  • Component:
    • Component creation:
      • Via ComponentSupport module(need Component.js and manifest.json)
      • Via data-sap-ui-oninit module and ComponentContainer(need Component.js and manifest.json or additional metadata)
    • Initial component instantiation:
      • Via createContent method
      • Via init method
